为什么 GitHub Pages 无法托管完整应用？

本页面是静态落地页；真正的 Builder 是动态全栈服务

🖥️

需要 Python 后端

FastAPI 服务处理聊天、SSE 推送、会话状态与 API 路由。GitHub Pages 仅支持静态 HTML/CSS/JS 文件。

🎭

需要 Playwright + Chromium

截图、样式提取、保真度对比依赖无头浏览器。静态托管无法运行 Chromium 进程。

🤖

需要 Claude Agent SDK

Agent 循环调用 10 个 MCP 工具，需要本地 CLI 或 API Key。这些密钥不能放在公开静态站点中。

推荐体验方式

评审者 clone 仓库 → 一键脚本 → 浏览器打开 localhost:8000，即可体验完整流程：粘贴 URL、观看 Agent 工作、对话精修、查看 Compare / Insights。

可选：自行部署 Docker 到 Render/Fly 获得公网 URL（见 DEPLOY.md）。

本地一键运行

复制下面任意一条命令，终端执行后即可启动

⚡ 一键 clone + 安装 + 启动（推荐） 
git clone https://github.com/AndyUneducated/lumalabs.git && cd lumalabs && chmod +x run.sh && ./run.sh
已有仓库时，直接启动 
./run.sh  # 自动创建 .venv、安装依赖、启动 http://localhost:8000

分步说明（若一键命令失败）

1

克隆仓库

git clone https://github.com/AndyUneducated/lumalabs.git && cd lumalabs

2

环境要求

依赖版本
Python3.10+
Git任意近期版本
Anthropic.env 中 API Key，或 claude login
Chromiumrun.sh 会自动安装（Playwright）
3

配置 API Key

cp .env.example .env，编辑 .env 填入 ANTHROPIC_API_KEY，或使用 Claude Code CLI 登录。

4

启动服务

chmod +x run.sh && ./run.sh → 浏览器打开 http://localhost:8000

5

体验流程

粘贴目标 URL（如 stripe.com）→ 等待 Agent 构建 → 在聊天中精修 → 切换 Preview / Compare / Insights 查看结果。

6

验证安装（可选）

python scripts/verify_phase7.py — 测试可扩展 job queue API
