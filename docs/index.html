Why GitHub Pages can't host the full app

This page is static; the real Builder is a dynamic full-stack service

🖥️

Python backend required

FastAPI handles chat, SSE, sessions, and API routes. GitHub Pages only serves static HTML/CSS/JS.

🎭

Playwright + Chromium

Screenshots, style extraction, and fidelity compare need a headless browser. Static hosting can't run Chromium.

🤖

Claude Agent SDK

The agent loop calls 10 MCP tools and needs a local CLI or API key — secrets can't live on a public static site.

Recommended for reviewers

Clone the repo → run ./run.sh → open http://localhost:8000 to try the full flow: paste a URL, watch the agent work, refine in chat, and use Compare / Insights.

Optional: deploy Docker to Render/Fly for a public API URL — see DEPLOY.md.
